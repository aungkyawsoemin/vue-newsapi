# vue-newsapi by Aung Kyaw Soe Min

## Project setup
```
npm install
```

### Setup .env variables then assign API_KEY. You can generate new API_KEY [here](https://newsapi.org/register)
```
cp .env.example .env
```

### Compiles and hot-reloads for development
```
npm run serve
```
